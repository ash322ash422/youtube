# Finance AI App

A starter project layout for a finance-focused AI application with frontend pages and backend services.

## Structure

- `app/`: frontend application files and reusable UI components
- `backend/`: services, AI modules, models, data folders, and utilities

## Run

```bash
python -m pip install -r requirements.txt
python -m streamlit run .\finance_ai_app\app\main.py 
```
