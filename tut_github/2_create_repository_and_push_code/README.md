
# GIT: How to push local code on remote git 

 ## After you have downloaded git and created git account (ONE TIME PROCESS)

We are going to create a repository with the name **gemini** (or choose your own name)

# PART A

### step1:  Open GitHub

Go to https://github.com 
 and log in.

### Step 2: Create a new repository

Click ➕ (top-right corner)

Click New repository

### Step 3: Fill repository details

Repository name: gemini

Description (optional):


Choose Public or Private


DO NOT check: 
- “Add a README” 
- “Add .gitignore”
- “Add license”

Click **Create Repository**

You will see new page displaying instructions on how to **create a new repository on the command line**


# Part B

 ### step1: Open your project folder (local machine)

cd path/to/your/project/gemini

### step2: Follow instruction as displayed on the git window


### step3: After adding new code:

git add .

git commit -m "commit"

git push -u origin main
