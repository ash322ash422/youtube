```python
import random

def calculate_pi(num_samples):
    inside_circle = 0

    for _ in range(num_samples):
        x, y = random.random(), random.random()
        if x**2 + y**2 <= 1:
            inside_circle += 1

    return 4 * inside_circle / num_samples

if __name__ == "__main__":
    num_samples = 1000000  # Adjust for precision
    pi_value = calculate_pi(num_samples)
    print(f"Estimated value of PI: {pi_value:.4f}")
```

### Explanation:
- The script uses the Monte Carlo method to estimate π.
- It generates random points in the unit square (0 to 1 for both x and y).
- It counts how many points fall inside the quarter circle (where \(x^2 + y^2 \leq 1\)).
- The ratio of points inside the circle to the total number of points, multiplied by 4, gives the estimate of π.
- The number of samples can be adjusted for better precision.
